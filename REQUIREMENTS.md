# Requirements

## Baseline reviewer path

- Python 3.10 or newer.
- Linux, macOS, or Windows with a POSIX-like shell for `scripts/run_all.sh`.
- No mandatory non-standard Python packages are required for the committed scripts and tests.
- No extra binary analysis tools are required.

## Optional scientific stack

`requirements.txt` lists common scientific packages for users who want to extend the artifact with notebook-style analysis or alternative plotting. These packages are optional for the baseline reproducibility commands in `ARTIFACT.md`.
